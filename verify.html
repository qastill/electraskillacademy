Verifikasi Sertifikat

Cek keaslian sertifikat Electra Skill Academy. Masukkan ID sertifikat (tertulis di pojok kanan-bawah PDF) atau scan QR code.

ℹ Cara verifikasi sertifikat:
  1. Buka file PDF sertifikat dari pemegang.
  2. ID sertifikat tercantum di pojok kanan-bawah (format ESA-LX-XXX-XXXXXXXX).
  3. Atau scan QR code yang tertera di sertifikat dengan smartphone — akan otomatis arahkan ke halaman ini dengan ID terisi.
  4. Klik Verifikasi. Hasil tampil real-time dari database resmi Electra Skill Academy.
